## 資料夾結構及翻譯

    .
    ├───Course Materials                      #課程資料
    │   ├─── Additional Practice              #課後輔導(練習)

    │   │         ├─── Extension-X            #第X個額外題目
    │   │         └─── Question-X(Homework)   #第X個題目(回家作業)

    │   │    └─── Week-X                      #第X週
    │   │         └─── Class-X                #第X課

    │   │    └─── Week-X                      #第X週
    │   │         └─── Homework-X             #第X個作業
    │   ├─── Quizzes                          #測驗
    │   │    └─── Week-X                      #第X週
    │   │         └─── Examination-X          #試卷X
    │   └─── Supplementary Materials          #補充資料

    │        └─── Other Materials             #其他資料
    │
    ├─── Extensions                           #課堂延伸
